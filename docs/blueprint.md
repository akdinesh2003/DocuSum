# **App Name**: DocuSense

## Core Features:

- File Support: Supports reading content from `.pdf` and `.txt` files.
- Batch Processing: Summarize one or many documents from the command line
- Quick Summary: Provide a concise summary as a bullet-point overview of key ideas (Quick Summary)
- Deep Summary: Provide a full summary as paragraph-style synthesis with tone and sentiment analysis (Deep Summary)
- Summary Quality Scoring: The AI tool will score summary quality based on length, clarity, and tone
- Sentiment Meter: Visually shows sentiment with a meter (positive/neutral/negative).
- Export Summary: Saves summaries to `.md` or `.txt` formats.

## Style Guidelines:

- Primary color: Soft teal (#76D7C4), evocative of clarity and intelligence.
- Background color: Light off-white (#F8F9FA) for readability and to avoid eye strain during long sessions.
- Accent color: Muted indigo (#8E44AD) to highlight key elements without being too distracting.
- Body and headline font: 'PT Sans' (sans-serif), which has a modern look and a little warmth or personality
- Code font: 'Source Code Pro' (monospace) for displaying any code snippets.
- Minimalist line icons that are related to documents, summarizing, and processing.
- Subtle transitions and animations for a smooth and responsive user experience, such as loading spinners during document processing.
